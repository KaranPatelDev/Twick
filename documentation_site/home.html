üöÄ Installation Guide

Prerequisites: Ensure you have Python 3.12+, Git, and optionally Docker installed on your system.

Method 1: Local Development Setup

Follow these steps to set up Twick for local development:

1. Clone the Repository

git clone https://github.com/yourusername/twick.git
cd twick

2. Create Virtual Environment

# Windows
python -m venv venv
venv\Scripts\activate

# macOS/Linux
python3 -m venv venv
source venv/bin/activate

3. Install Dependencies

pip install -r requirements.txt

4. Database Setup

python manage.py makemigrations
python manage.py migrate

5. Create Superuser

python manage.py createsuperuser

6. Run Development Server

python manage.py runserver

Visit http://127.0.0.1:8000 to access your local Twick instance!

Nginx Configuration

Sample Nginx configuration for production:

server {
    listen 80;
    server_name yourdomain.com www.yourdomain.com;
    
    location / {
        proxy_pass http://127.0.0.1:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
    
    location /static/ {
        alias /var/www/static/;
        expires 1y;
        add_header Cache-Control "public, immutable";
    }
    
    location /media/ {
        alias /var/www/media/;
        expires 1y;
        add_header Cache-Control "public";
    }
}

üåê Deploy Documentation to Netlify

This documentation site is ready for deployment on Netlify. Follow these steps:

Method 1: Git-based Deployment

  1. Push your project to GitHub/GitLab
  2. Connect your repository to Netlify
  3. Set build command: echo "Static site, no build needed"
  4. Set publish directory: documentation_site
  5. Deploy!

Method 2: Manual Deployment

  1. Zip the documentation_site folder
  2. Go to Netlify
  3. Drag and drop your zip file
  4. Your documentation is live!

Netlify Configuration

Create a netlify.toml file in the documentation_site folder:

[build]
  publish = "."
  
[[redirects]]
  from = "/api/*"
  to = "/pages/api-reference.html"
  status = 200

[[headers]]
  for = "/*"
  [headers.values]
    X-Frame-Options = "DENY"
    X-XSS-Protection = "1; mode=block"
Ready to Deploy! Your documentation site is optimized for static hosting and will work perfectly on Netlify, GitHub Pages, or any static hosting service.
