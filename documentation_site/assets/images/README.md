# Documentation Site Images

This folder contains images used in the Twick documentation site.

## Structure

- `favicon.svg` - Site favicon
- `logo.png` - Twick logo
- `screenshots/` - Application screenshots
- `diagrams/` - Architecture diagrams
- `features/` - Feature demonstration images

## Image Guidelines

- Use modern web formats (WebP, SVG when possible)
- Optimize images for web (compression)
- Use descriptive filenames
- Include alt text for accessibility
